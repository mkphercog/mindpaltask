# MindPal Task #2

1. clone this repo
2. use `npm install`
3. after that create `.env` with:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_SERVICE_ROLE_KEY`
- `VITE_ENCRYPTION_KEY`

4. now you are ready to launch app by `npm run dev`
5. try add some record to your supabase db
6. you will see new record on the list above form
